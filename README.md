#### Migranti nel Salento
Data driven project based on a workshop held in Lecce at June 20, 2016 - dataninja / ondata
[Click for the live version here](http://ondata.github.io/migranti-lecce)

#### Project based on > [Start Bootstrap](http://startbootstrap.com/) - [Grayscale](http://startbootstrap.com/template-overviews/grayscale/)

[Grayscale](http://startbootstrap.com/template-overviews/grayscale/) is a multipurpose, one page HTML theme for [Bootstrap](http://getbootstrap.com/) created by [Start Bootstrap](http://startbootstrap.com/). This template features various content sections and a Google Maps section with a custom map marker.
